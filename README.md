# OSINT-AI
OSINT AI — профессиональный OSINT-фреймворк для сбора данных из открытых источников. Сканирование портов, поиск по email/username, веб-анализ, SSL, DNS, BGP. Модульная архитектура с поддержкой модов и API. Многопоточность, экспорт отчётов, русский/английский интерфейс. Этичный инструмент для кибербезопасности и расследований.
